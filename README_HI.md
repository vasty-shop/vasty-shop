## Vasty Shop क्या है?

Vasty Shop एक ओपन-सोर्स मल्टी-वेंडर ई-कॉमर्स मार्केटप्लेस प्लेटफ़ॉर्म है। Amazon, Shopify या Etsy जैसा अपना मार्केटप्लेस बनाएं — AI-संचालित सिफारिशें, Stripe Connect के ज़रिए वेंडरों को स्वचालित भुगतान, फ्लैश सेल, POS सिस्टम और डिलीवरी प्रबंधन के साथ — सब कुछ सेल्फ-होस्ट योग्य।

## विक्रेता डैशबोर्ड

हर विक्रेता को एक सेल्फ-सर्विस कंट्रोल पैनल मिलता है जिसमें रीयल-टाइम KPI (राजस्व, ऑर्डर, उत्पाद, ग्राहक), आय विवरण — सकल बिक्री, डिलीवरी लागत, शुद्ध लाभ — और पूर्ण ऑर्डर, उत्पाद तथा अनुमोदन प्रबंधन शामिल है।

## त्वरित प्रारंभ

### Docker (अनुशंसित)

> **पूर्वापेक्षाएँ**: [Docker](https://docs.docker.com/get-docker/) और [Docker Compose](https://docs.docker.com/compose/install/)

```bash
git clone https://github.com/vasty-shop/vasty-shop.git
cd vasty-shop

# 1. उदाहरणों से env फ़ाइलें बनाएँ
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env

# 2. (वैकल्पिक) सेटअप विज़ार्ड चलाएँ
docker compose --profile setup run --rm setup

# 3. सभी सेवाएँ शुरू करें (PostgreSQL, Redis, Backend, Frontend)
docker compose up --build

# 4. डेटाबेस माइग्रेशन चलाएँ (नए टर्मिनल में)
docker compose exec backend npm run migrate

# 5. (वैकल्पिक) डेटाबेस सीड करें
docker compose exec backend npm run seed
```

ऐप इन पते पर उपलब्ध होगा:

| सेवा | URL |
|---------|-----|
| **Frontend** | http://localhost:5186 |
| **Backend API** | http://localhost:4005/api/v1 |
| **API डॉक्स (Swagger)** | http://localhost:4005/api/v1/docs |
| **हेल्थ (प्रोवाइडर्स)** | http://localhost:4005/api/v1/health/providers |
| **WebSocket** | http://localhost:3002 |

> **नोट:** प्रोडक्शन में एडमिन पासवर्ड तुरंत बदलें।

#### Docker सेवाएँ

| सेवा | इमेज | पोर्ट |
|---------|-------|------|
| **PostgreSQL** | postgres:16-alpine | 5433 |
| **Redis** | redis:7-alpine | 6379 |
| **Backend** | node:20-alpine (NestJS) | 4005, 3002 |
| **Frontend** | node:20-alpine (Vite) | 5186 |

#### उपयोगी कमांड

```bash
# सभी सेवाएँ रोकें
docker compose down

# रोकें और सारा डेटा हटाएँ
docker compose down -v

# backend लॉग देखें
docker compose logs -f backend

# माइग्रेशन चलाएँ
docker compose exec backend npm run migrate

# डेटाबेस सीड करें
docker compose exec backend npm run seed

# PostgreSQL शेल खोलें
docker compose exec postgres psql -U postgres -d vasty_shop_dev

# सेटअप विज़ार्ड चलाएँ
docker compose --profile setup run --rm setup

# वैकल्पिक सेवाओं के साथ शुरू करें (जैसे Meilisearch, MinIO)
docker compose --profile meilisearch --profile minio up -d
```

### लोकल डेवलपमेंट (Docker के बिना)

> **पूर्वापेक्षाएँ**: Node.js 20+, PostgreSQL 16+, Redis 7+

```bash
git clone https://github.com/vasty-shop/vasty-shop.git
cd vasty-shop

# Backend
cd backend
cp .env.example .env
# .env संपादित करें: DATABASE_HOST=localhost और REDIS_HOST=localhost सेट करें
npm install
npm run migrate
npm run start:dev

# Frontend (नया टर्मिनल)
cd frontend
cp .env.example .env
npm install --legacy-peer-deps
npm run dev
```

## विशेषताएँ

### ई-कॉमर्स कोर
- **उत्पाद** -- वेरिएंट, गुण, इन्वेंटरी, डिजिटल उत्पाद, बल्क इम्पोर्ट
- **ऑर्डर** -- मल्टी-वेंडर ऑर्डर स्प्लिटिंग, स्टेटस ट्रैकिंग, रिफंड
- **कार्ट** -- पर्सिस्टेंट कार्ट, गेस्ट चेकआउट, मल्टी-करेंसी
- **श्रेणियाँ** -- फ़िल्टर और खोज के साथ नेस्टेड श्रेणियाँ
- **समीक्षाएँ** -- रेटिंग, फ़ोटो, सत्यापित खरीद बैज

### भुगतान और वित्त
- **Stripe Connect** -- प्लेटफ़ॉर्म शुल्क के साथ स्वचालित वेंडर पेआउट
- **PayPal** -- वैकल्पिक भुगतान गेटवे
- **वॉलेट** -- टॉप-अप और खर्च के साथ ग्राहक वॉलेट
- **एस्क्रो** -- डिलीवरी तक सुरक्षित भुगतान होल्ड
- **वितरण** -- थोक वेंडर पेआउट, कमीशन ट्रैकिंग
- **व्यय** -- वेंडरों के लिए व्यापार व्यय ट्रैकिंग

### विपणन और विकास
- **फ्लैश सेल** -- काउंटडाउन के साथ समय-सीमित सौदे
- **अभियान** -- शेड्यूलिंग के साथ प्रचार अभियान
- **कूपन** -- प्रतिशत, निश्चित, मुफ़्त शिपिंग
- **गिफ्ट कार्ड** -- बैलेंस के साथ डिजिटल गिफ्ट कार्ड
- **लॉयल्टी** -- कैशबैक के साथ अंक प्रणाली
- **रेफरल** -- ग्राहक रेफरल कार्यक्रम
- **सर्ज मूल्य** -- माँग के आधार पर गतिशील मूल्य

### संचालन
- **POS** -- बारकोड स्कैनिंग के साथ बिक्री बिंदु
- **डिलीवरी** -- ज़ोन-आधारित मूल्य, ट्रैकिंग, डिलीवरी पार्टनर
- **पार्सल** -- पार्सल डिलीवरी प्रबंधन
- **कर** -- क्षेत्र के अनुसार कर गणना
- **निर्यात** -- CSV/Excel डेटा निर्यात

### प्लेटफ़ॉर्म
- **17 भाषाएँ** -- AR, BN, DE, EN, ES, FR, HI, ID, IT, JA, KO, MS, PT, RU, TR, UR, ZH
- **AI** -- उत्पाद सिफारिशें, स्मार्ट खोज
- **Blog/CMS** -- कंटेंट प्रबंधन
- **चैट** -- ग्राहक-विक्रेता रीयल-टाइम मैसेजिंग
- **सूचनाएँ** -- ईमेल, WebSocket, पुश
- **एडमिन डैशबोर्ड** -- पूर्ण प्लेटफ़ॉर्म एनालिटिक्स

## परियोजना संरचना

```
vasty-shop/
├── backend/              # NestJS API (69 मॉड्यूल, 80+ तालिकाएँ)
│   ├── src/modules/      # products, orders, cart, payments, delivery,
│   │                     # campaigns, coupons, flash-sales, gift-cards,
│   │                     # loyalty, referral, pos, ai, blog, chat, ...
│   └── migrations/       # PostgreSQL माइग्रेशन
├── frontend/             # React + Vite + Tailwind (17 भाषाएँ)
├── shared/               # साझा प्रकार और उपयोगिताएँ
└── .github/workflows/    # CI/CD
```
