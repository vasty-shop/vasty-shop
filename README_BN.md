## Vasty Shop কী?

Vasty Shop হলো একটি ওপেন-সোর্স মাল্টি-ভেন্ডর ই-কমার্স মার্কেটপ্লেস প্ল্যাটফর্ম। Amazon, Shopify বা Etsy-এর মতো নিজের মার্কেটপ্লেস তৈরি করুন AI-চালিত রিকমেন্ডেশন, Stripe Connect-এর মাধ্যমে স্বয়ংক্রিয় ভেন্ডর পেআউট, ফ্ল্যাশ সেল, POS সিস্টেম এবং ডেলিভারি ম্যানেজমেন্ট সহ — সবকিছু সেলফ-হোস্টেবল।

## দ্রুত শুরু

### Docker (প্রস্তাবিত)

> **পূর্বশর্ত**: [Docker](https://docs.docker.com/get-docker/) এবং [Docker Compose](https://docs.docker.com/compose/install/)

```bash
git clone https://github.com/vasty-shop/vasty-shop.git
cd vasty-shop

# 1. উদাহরণ থেকে env ফাইল তৈরি করুন
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env

# 2. (ঐচ্ছিক) প্রোভাইডার বাছাই করতে সেটআপ উইজার্ড চালান
docker compose --profile setup run --rm setup

# 3. সব সার্ভিস শুরু করুন (PostgreSQL, Redis, Backend, Frontend)
docker compose up --build

# 4. ডাটাবেস মাইগ্রেশন চালান (নতুন টার্মিনালে)
docker compose exec backend npm run migrate

# 5. (ঐচ্ছিক) ডাটাবেস সিড করুন
docker compose exec backend npm run seed
```

অ্যাপ নিম্নলিখিত ঠিকানায় উপলব্ধ:

| সার্ভিস | URL |
|---------|-----|
| **Frontend** | http://localhost:5186 |
| **Backend API** | http://localhost:4005/api/v1 |
| **API Docs (Swagger)** | http://localhost:4005/api/v1/docs |
| **Health (প্রোভাইডার)** | http://localhost:4005/api/v1/health/providers |
| **WebSocket** | http://localhost:3002 |

#### ডিফল্ট অ্যাডমিন ক্রেডেনশিয়াল

| ফিল্ড | মান |
|-------|-------|
| **ইমেইল** | `admin@vasty.shop` |
| **পাসওয়ার্ড** | `admin123` |

> **নোট:** প্রোডাকশনে অ্যাডমিন পাসওয়ার্ড অবিলম্বে পরিবর্তন করুন।

#### Docker সার্ভিস

| সার্ভিস | ইমেজ | পোর্ট |
|---------|-------|------|
| **PostgreSQL** | postgres:16-alpine | 5432 |
| **Redis** | redis:7-alpine | 6379 |
| **Backend** | node:20-alpine (NestJS) | 4005, 3002 |
| **Frontend** | node:20-alpine (Vite) | 5186 |

#### দরকারি কমান্ড

```bash
# সব সার্ভিস বন্ধ করুন
docker compose down

# বন্ধ করুন এবং সব ডেটা মুছুন
docker compose down -v

# backend লগ দেখুন
docker compose logs -f backend

# মাইগ্রেশন চালান
docker compose exec backend npm run migrate

# ডাটাবেস সিড
docker compose exec backend npm run seed

# PostgreSQL শেল অ্যাক্সেস
docker compose exec postgres psql -U postgres -d vasty_shop_dev

# সেটআপ উইজার্ড চালান
docker compose --profile setup run --rm setup

# ঐচ্ছিক সার্ভিসসহ শুরু (যেমন Meilisearch, MinIO)
docker compose --profile meilisearch --profile minio up -d
```

### লোকাল ডেভেলপমেন্ট (Docker ছাড়া)

> **পূর্বশর্ত**: Node.js 20+, PostgreSQL 16+, Redis 7+

```bash
git clone https://github.com/vasty-shop/vasty-shop.git
cd vasty-shop

# Backend
cd backend
cp .env.example .env
# .env সম্পাদনা করুন: DATABASE_HOST=localhost এবং REDIS_HOST=localhost সেট করুন
npm install
npm run migrate
npm run start:dev

# Frontend (নতুন টার্মিনাল)
cd frontend
cp .env.example .env
npm install --legacy-peer-deps
npm run dev
```

## বৈশিষ্ট্য

### ই-কমার্স কোর
- **প্রোডাক্ট** -- ভেরিয়েন্ট, অ্যাট্রিবিউট, ইনভেন্টরি, ডিজিটাল প্রোডাক্ট, বাল্ক ইম্পোর্ট
- **অর্ডার** -- মাল্টি-ভেন্ডর অর্ডার বিভাজন, স্ট্যাটাস ট্র্যাকিং, রিফান্ড
- **কার্ট** -- স্থায়ী কার্ট, গেস্ট চেকআউট, মাল্টি-কারেন্সি
- **ক্যাটাগরি** -- ফিল্টার এবং সার্চসহ নেস্টেড ক্যাটাগরি
- **রিভিউ** -- রেটিং, ছবি, যাচাইকৃত কেনার ব্যাজ

### পেমেন্ট ও অর্থ
- **Stripe Connect** -- প্ল্যাটফর্ম ফিসহ স্বয়ংক্রিয় ভেন্ডর পেআউট
- **PayPal** -- বিকল্প পেমেন্ট গেটওয়ে
- **ওয়ালেট** -- টপ-আপ ও খরচসহ গ্রাহক ওয়ালেট
- **এসক্রো** -- ডেলিভারির আগে পর্যন্ত নিরাপদ পেমেন্ট হোল্ড
- **ডিসবার্সমেন্ট** -- বাল্ক ভেন্ডর পেআউট, কমিশন ট্র্যাকিং
- **ব্যয়** -- ভেন্ডরদের জন্য ব্যবসায়িক ব্যয় ট্র্যাকিং

### মার্কেটিং ও প্রবৃদ্ধি
- **ফ্ল্যাশ সেল** -- কাউন্টডাউনসহ সময়-সীমিত অফার
- **ক্যাম্পেইন** -- সময়সূচীসহ প্রমোশনাল ক্যাম্পেইন
- **কুপন** -- শতাংশ, নির্দিষ্ট পরিমাণ, ফ্রি শিপিং
- **গিফট কার্ড** -- ব্যালেন্সসহ ডিজিটাল গিফট কার্ড
- **লয়ালটি** -- ক্যাশব্যাকসহ পয়েন্ট সিস্টেম
- **রেফারেল** -- গ্রাহক রেফারেল প্রোগ্রাম
- **সার্জ প্রাইসিং** -- চাহিদা-ভিত্তিক গতিশীল মূল্য

### অপারেশন
- **POS** -- বারকোড স্ক্যানিংসহ পয়েন্ট-অফ-সেল
- **ডেলিভারি** -- জোন-ভিত্তিক মূল্য, ট্র্যাকিং, ডেলিভারি পার্টনার
- **পার্সেল** -- পার্সেল ডেলিভারি ম্যানেজমেন্ট
- **ট্যাক্স** -- অঞ্চল অনুযায়ী ট্যাক্স গণনা
- **এক্সপোর্ট** -- CSV/Excel ডেটা এক্সপোর্ট

### প্ল্যাটফর্ম
- **১৭টি ভাষা** -- AR, BN, DE, EN, ES, FR, HI, ID, IT, JA, KO, MS, PT, RU, TR, UR, ZH
- **AI** -- প্রোডাক্ট রিকমেন্ডেশন, স্মার্ট সার্চ
- **Blog/CMS** -- কন্টেন্ট ম্যানেজমেন্ট
- **চ্যাট** -- গ্রাহক-ভেন্ডর রিয়েল-টাইম মেসেজিং
- **নোটিফিকেশন** -- ইমেইল, WebSocket, পুশ
- **অ্যাডমিন ড্যাশবোর্ড** -- সম্পূর্ণ প্ল্যাটফর্ম অ্যানালিটিক্স

## প্রজেক্ট স্ট্রাকচার

```
vasty-shop/
├── backend/              # NestJS API (৫৩ মডিউল, ৬৭ টেবিল)
│   ├── src/modules/      # products, orders, cart, payments, delivery,
│   │                     # campaigns, coupons, flash-sales, gift-cards,
│   │                     # loyalty, referral, pos, ai, blog, chat, ...
│   └── migrations/       # PostgreSQL মাইগ্রেশন
├── frontend/             # React + Vite + Tailwind (১৭টি ভাষা)
├── shared/               # শেয়ারড টাইপ ও ইউটিলিটি
└── .github/workflows/    # CI/CD
```
