Shop Store Manual Test

Open the browser console and run the following commands to test the shop store:

// Import the test functions (if in dev mode)
import { runAllShopStoreTests } from './src/stores/useShopStore.test.ts';

// Or if tests are available on window object:
window.shopStoreTests.runAll();

// Individual tests:
window.shopStoreTests.testSetCurrentShop();
window.shopStoreTests.testSetShops();
window.shopStoreTests.testSwitchShop();
window.shopStoreTests.testSwitchToInvalidShop();
window.shopStoreTests.testClearShopContext();

Manual Testing Steps

  1. Start the development server: npm run dev
  2. Open the browser console (F12)
  3. Test the store functions: 
    // Get the store
import { useShopStore } from './src/stores/useShopStore';
const store = useShopStore.getState();

// Test 1: Set shops
const mockShops = [
  {
    id: '1',
    name: 'Test Shop 1',
    slug: 'test-shop-1',
    status: 'active',
    isVerified: true
  }
];
store.setShops(mockShops);
console.log('Shops:', useShopStore.getState().shops);

// Test 2: Set current shop
store.setCurrentShop(mockShops[0]);
console.log('Current shop:', useShopStore.getState().currentShop);

// Test 3: Switch shop
store.switchShop('1');
console.log('After switch:', useShopStore.getState().currentShop);

// Test 4: Clear context
store.clearShopContext();
console.log('After clear:', useShopStore.getState());

Integration Testing

  1. Login as a vendor
  2. Check that shop context is set after login
  3. Verify API requests include x-shop-id header
  4. Test shop switching functionality
  5. Verify logout clears shop context
