## Sub-Components Detail

### 1. AnimatedCounter
**Purpose**: Animates numbers from 0 to target value
**Props**:
- `value: number` - Target number
- `duration?: number` - Animation duration (default: 2s)

**Features**:
- Easing function for smooth animation
- Viewport detection (animates only when visible)
- RequestAnimationFrame for performance

### 2. ValueCard
**Purpose**: Display core company values
**Props**:
- `icon: React.ElementType` - Lucide icon component
- `title: string` - Value title
- `description: string` - Value description
- `delay: number` - Animation delay

**Features**:
- Gradient icon background
- Hover effects (scale, border)
- Scroll-triggered animation

### 3. StatCard
**Purpose**: Display animated statistics
**Props**:
- `value: number` - Number to display
- `label: string` - Description label
- `suffix?: string` - Text after number (e.g., "+", "M+")
- `prefix?: string` - Text before number (e.g., "$")
- `delay: number` - Animation delay

**Features**:
- Gradient text color
- Animated counter integration
- Viewport-based triggering

### 4. TeamMember
**Purpose**: Display team member profile
**Props**:
- `name: string` - Member name
- `role: string` - Job title
- `image: string` - Photo URL
- `delay: number` - Animation delay

**Features**:
- Image hover zoom effect
- Card shadow elevation
- Smooth entrance animation

### 5. SustainabilityFeature
**Purpose**: List sustainability initiatives
**Props**:
- `icon: React.ElementType` - Feature icon
- `title: string` - Feature title
- `description: string` - Feature description
- `delay: number` - Animation delay

**Features**:
- Icon in colored circle
- Horizontal layout
- Slide-in animation

## Animation Timeline

### On Page Load
1. Hero content fades in (0-0.6s)
2. Hero image scales up (0.4-1.2s)

### On Scroll (Sequential)
1. Story section slides up
2. Values cards stagger in (100ms apart)
3. Stats counters animate when visible
4. Team members appear sequentially
5. Sustainability features slide in
6. CTA section fades in

### Continuous Animations
1. Hero background elements rotate
2. Rating badge floats up/down
3. Eco badge floats with rotation
4. CTA background pattern moves
