## Vasty Shop کیا ہے؟

Vasty Shop ایک اوپن سورس ملٹی وینڈر ای کامرس مارکیٹ پلیس پلیٹ فارم ہے۔ Amazon، Shopify یا Etsy کی طرز پر اپنا مارکیٹ پلیس بنائیں — AI سے چلنے والی سفارشات، Stripe Connect کے ذریعے وینڈرز کو خودکار ادائیگیاں، فلیش سیل، POS سسٹم اور ڈیلیوری مینجمنٹ کے ساتھ — سب کچھ سیلف ہوسٹ ہو سکتا ہے۔

## فوری آغاز

### Docker (تجویز کردہ)

> **درکار**: [Docker](https://docs.docker.com/get-docker/) اور [Docker Compose](https://docs.docker.com/compose/install/)

```bash
git clone https://github.com/vasty-shop/vasty-shop.git
cd vasty-shop

# 1. مثالوں سے env فائلیں بنائیں
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env

# 2. (اختیاری) پرووائیڈرز انٹرایکٹو طور پر منتخب کرنے کے لیے سیٹ اپ وزرڈ چلائیں
docker compose --profile setup run --rm setup

# 3. تمام خدمات شروع کریں (PostgreSQL, Redis, Backend, Frontend)
docker compose up --build

# 4. ڈیٹا بیس مائیگریشن چلائیں (نئے ٹرمینل میں)
docker compose exec backend npm run migrate

# 5. (اختیاری) ڈیٹا بیس میں ڈیٹا شامل کریں
docker compose exec backend npm run seed
```

ایپ درج ذیل پتوں پر دستیاب ہوگی:

| خدمت | URL |
|---------|-----|
| **Frontend** | http://localhost:5186 |
| **Backend API** | http://localhost:4005/api/v1 |
| **API دستاویزات (Swagger)** | http://localhost:4005/api/v1/docs |
| **ہیلتھ (پرووائیڈرز)** | http://localhost:4005/api/v1/health/providers |
| **WebSocket** | http://localhost:3002 |

#### ڈیفالٹ ایڈمن کریڈنشلز

| فیلڈ | قدر |
|-------|-------|
| **ای میل** | `admin@vasty.shop` |
| **پاس ورڈ** | `admin123` |

> **نوٹ:** پروڈکشن میں ایڈمن پاس ورڈ فوری طور پر تبدیل کریں۔

#### Docker خدمات

| خدمت | امیج | پورٹ |
|---------|-------|------|
| **PostgreSQL** | postgres:16-alpine | 5433 |
| **Redis** | redis:7-alpine | 6379 |
| **Backend** | node:20-alpine (NestJS) | 4005, 3002 |
| **Frontend** | node:20-alpine (Vite) | 5186 |

#### مفید کمانڈز

```bash
# تمام خدمات بند کریں
docker compose down

# بند کریں اور تمام ڈیٹا حذف کریں
docker compose down -v

# backend لاگز دیکھیں
docker compose logs -f backend

# مائیگریشن چلائیں
docker compose exec backend npm run migrate

# ڈیٹا بیس سیڈ کریں
docker compose exec backend npm run seed

# PostgreSQL شیل تک رسائی
docker compose exec postgres psql -U postgres -d vasty_shop_dev

# سیٹ اپ وزرڈ چلائیں
docker compose --profile setup run --rm setup

# اختیاری خدمات کے ساتھ شروع کریں (جیسے Meilisearch, MinIO)
docker compose --profile meilisearch --profile minio up -d
```

### لوکل ڈیولپمنٹ (Docker کے بغیر)

> **درکار**: Node.js 20+, PostgreSQL 16+, Redis 7+

```bash
git clone https://github.com/vasty-shop/vasty-shop.git
cd vasty-shop

# Backend
cd backend
cp .env.example .env
# .env میں ترمیم کریں: DATABASE_HOST=localhost اور REDIS_HOST=localhost سیٹ کریں
npm install
npm run migrate
npm run start:dev

# Frontend (نیا ٹرمینل)
cd frontend
cp .env.example .env
npm install --legacy-peer-deps
npm run dev
```

## خصوصیات

### ای کامرس کور
- **مصنوعات** -- ویریئنٹس، خصوصیات، انوینٹری، ڈیجیٹل مصنوعات، بلک امپورٹ
- **آرڈرز** -- ملٹی وینڈر آرڈر تقسیم، اسٹیٹس ٹریکنگ، ری فنڈ
- **کارٹ** -- مستقل کارٹ، بطور مہمان چیک آؤٹ، ملٹی کرنسی
- **زمرہ جات** -- فلٹرز اور تلاش کے ساتھ نیسٹڈ زمرہ جات
- **جائزے** -- درجہ بندی، تصاویر، تصدیق شدہ خریداری بیجز

### ادائیگی اور مالیات
- **Stripe Connect** -- پلیٹ فارم فیس کے ساتھ خودکار وینڈر ادائیگیاں
- **PayPal** -- متبادل ادائیگی گیٹ وے
- **والٹ** -- ٹاپ اپ اور خرچ کے ساتھ گاہک والٹ
- **اسکرو** -- ڈیلیوری تک محفوظ ادائیگی
- **تقسیم** -- بلک وینڈر ادائیگیاں، کمیشن ٹریکنگ
- **اخراجات** -- وینڈرز کے لیے کاروباری اخراجات ٹریکنگ

### مارکیٹنگ اور ترقی
- **فلیش سیل** -- کاؤنٹ ڈاؤن کے ساتھ محدود وقت کے معاہدے
- **مہمات** -- شیڈولنگ کے ساتھ پروموشنل مہمات
- **کوپنز** -- فیصد، مقررہ رقم، مفت شپنگ
- **گفٹ کارڈز** -- بیلنس کے ساتھ ڈیجیٹل گفٹ کارڈز
- **وفاداری** -- کیش بیک کے ساتھ پوائنٹ سسٹم
- **ریفرل** -- گاہک ریفرل پروگرام
- **متحرک قیمت** -- طلب پر مبنی قیمتیں

### آپریشنز
- **POS** -- بارکوڈ اسکیننگ کے ساتھ پوائنٹ آف سیل
- **ڈیلیوری** -- زون کی بنیاد پر قیمت، ٹریکنگ، ڈیلیوری پارٹنرز
- **پارسل** -- پارسل ڈیلیوری مینجمنٹ
- **ٹیکس** -- خطے کے مطابق ٹیکس کا حساب
- **ایکسپورٹ** -- CSV/Excel ڈیٹا ایکسپورٹ

### پلیٹ فارم
- **17 زبانیں** -- AR, BN, DE, EN, ES, FR, HI, ID, IT, JA, KO, MS, PT, RU, TR, UR, ZH
- **AI** -- مصنوعات کی سفارشات، اسمارٹ تلاش
- **Blog/CMS** -- مواد کا انتظام
- **چیٹ** -- گاہک-وینڈر ریئل ٹائم میسجنگ
- **اطلاعات** -- ای میل، WebSocket، پش
- **ایڈمن ڈیش بورڈ** -- مکمل پلیٹ فارم تجزیات

## ٹیکنیکل اسٹیک

| پرت | ٹیکنالوجی |
|-------|------------|
| **Backend** | NestJS, TypeScript, PostgreSQL (raw SQL), Redis, Socket.io |
| **Frontend** | React, Vite, TypeScript, Tailwind CSS, Radix UI, i18next |
| **اسٹوریج** | پلگ ایبل: local-fs, S3, Cloudflare R2, MinIO, B2, GCS, Azure |
| **ادائیگیاں** | Stripe, Stripe Connect, PayPal |
| **AI** | OpenAI، Anthropic، Gemini، Groq، Ollama |
| **تلاش** | PostgreSQL (pg-trgm)، Meilisearch، Typesense |

```

```
